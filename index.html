Automotive Dealership CRUD
  • Home
  • Weather
  • Dealership

This single page application demonstrates the use of a CRUD (Create, Read, Update, Delete) system for managing an automotive dealership's inventory. The system interfaces with a MySQL relational database through a REST api created with PHP.

Technologies Used

  • CRUD (Create, Read, Update, Delete)
  • REST API
  • MySQL database
  • PHP
  • jQuery
  • AJAX
  • Bootstrap 4
